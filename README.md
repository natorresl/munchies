Munchies

This is a Next.js 13+ web application that displays restaurants with filtering by category, delivery time, and price range.
It fetches data from external APIs and dynamically updates restaurant cards based on query parameters in the URL.

Fetches restaurant data from API
✅ Open/Closed status per restaurant (live from API)
✅ Filtering by:
Category (e.g. Pizza, Vegan, etc.)
Delivery time (0–10 min, 10–30 min, etc.)
Price range ($, $$, $$$)
✅ URL-based filters (shareable links)
✅ Responsive design with Tailwind CSS
